=== custon Adobe AIR Badge ===
for use with blago AIR Badge plugin for Wordpress


Original AIR badge made by Grant Skinner. 
Download and SDK: http://www.adobe.com/devnet/air/articles/badge_for_air.html



To see a local demo of the javascript being called (not the click being counted), open 
test > testbadge.html and click on the "install" btn. This will trigger a javascript alert.
This will not work if you have not set your Global Security permissions to allow the flash to do things locally!
